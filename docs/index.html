CDDA-X

CD Digital Audio eXtractor

Download

Current version: v22.4.1 from March 19, 2024

  • Installer (.msi)
  • Portable version (.zip)
  • Source Code

Features

  • Can encode MP3 and FLAC files
  • Can read CD-Text
  • Can download metadata and cover art from MusicBrainz
  • Can rip hidden tracks
  • Is free software (GNU GPLv3)

System Requirements

Basic functionality works on .NET Framework 4.0 on Windows XP or higher. For MusicBrainz and High-DPI support, .NET Framework 4.8 is required. Windows 10 and 11 already ship with .NET 4.8.

NOTE: On Windows 7, youâ€™ll need to manually enable TLS 1.2 in order to be able to connect to MusicBrainz.

There is some experimental code to support Mono on Linux, but I recommend you run it in wine with real .NET 4.8 installed.
